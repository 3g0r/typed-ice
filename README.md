# TypeScript definitions for [Zeroc ICE](https://github.com/zeroc-ice/ice) [![npm version][npm-image]][npm-url] 

Typings are incomplete. New definitions are added on demand.

## Usage

*tsconfig.json*
```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "ice": ["node_modules/typed-ice"]
    }
  }
}
```
